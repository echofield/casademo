# Vendor DPA and Governance Template

Use this as a dossier checklist. Populate with customer-specific legal/process data.

## A. Data processing register (RoPA)
- Processing purpose(s)
- Categories of personal data
- Data subjects
- Recipients/subprocessors
- Retention windows
- Security measures

## B. Vendor/subprocessor map
- Supabase: role, region, transfer mechanism, DPA status
- Vercel: role, region, transfer mechanism, DPA status
- Any mail/notification provider: role, region, DPA status

## C. Contractual pack
- DPA execution status (controller/processor roles)
- SCC/transfer clauses where needed
- Security annex
- Incident notification obligations

## D. Operating procedures
- DSAR SOP (request intake, identity verification, SLA, evidence capture)
- Breach response SOP (detection, severity, 72-hour path)
- Access review cadence (quarterly or defined interval)
- Retention/deletion/anonymization policy owner

## E. Release governance
- Change approval owner
- Staging gate evidence link
- Production release sign-off
